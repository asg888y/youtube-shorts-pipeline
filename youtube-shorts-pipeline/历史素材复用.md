# 历史素材复用

> 避免重复生成，节省成本

---

## 素材库位置

```
/Users/yezhong/youtube-shorts-pipeline/local_assets/
├── images/     # 图片素材
├── audio/      # 音频素材
└── music/      # 背景音乐
```

### 图片素材

```
local_assets/images/
├── broll_0.png
├── broll_1.png
├── broll_2.png
└── ...（按需添加）
```

---

## 复用方式

### 方式一：全部使用历史素材

```
用户指令："使用历史素材"
```

系统直接从 `local_assets/images/` 读取图片，不调用API。

### 方式二：混合使用

```
用户指令："生成5张新图片，其余用历史素材"
```

系统生成5张新图片，剩余素材从历史库补充。

### 方式三：API失败自动补充

API生成失败时，自动从历史素材库补充。

## 素材分类管理

### 建议分类

```
local_assets/images/
├── military/       # 军事主题
├── tech/           # 科技主题
├── nature/         # 自然风景
├── city/           # 城市景观
├── abstract/       # 抽象背景
└── general/        # 通用素材
```

### 使用方式

```
用户指令："使用军事主题历史素材"
```

系统从 `local_assets/images/military/` 目录读取。

---

## 素材命名规范

### 文件名格式

```
{主题}_{序号}.png

示例：
military_001.png
military_002.png
tech_001.png
nature_001.png
```

### 元数据文件（可选）

每个分类目录可包含 `meta.json`：

```json
{
  "theme": "military",
  "count": 10,
  "description": "军事基地、武器、战场场景",
  "created": "2026-05-08"
}
```

## 注意事项

1. 历史素材需符合9:16竖屏比例
2. 素材质量需达到1080x1920分辨率
3. 定期清理低质量素材
4. 避免素材重复使用导致画面雷同
