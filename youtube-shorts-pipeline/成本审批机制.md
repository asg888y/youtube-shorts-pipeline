# 成本审批机制

> 所有素材生成必须经过审批，避免成本浪费

## 审批流程

### 1. 提交申请

生成素材前，系统显示成本预估：

```
素材生成申请：
- 图片：20张 × $0.05 = $1.00
- 视频：2个 × $0.2 = $0.40
- 预估总成本：$1.40

请回复"同意"确认生成，其他任何回复均取消：
```

### 2. 授权规则

- ✅ **唯一有效授权**：用户输入文字"同意"
- ❌ **无效授权**：任何其他字符、表情、ID审批等
- ❌ **自动执行**：禁止未经审批自动生成素材

### 3. 执行生成

收到"同意"后，系统执行素材生成。

### 4. 失败重试

如果API生成失败，需要重新申请：

```
素材生成失败，需要重试：
- 图片：3张失败 × $0.05 = $0.15
- 视频：1个失败 × $0.2 = $0.20
- 重试成本：$0.35

请回复"同意"确认重试，其他任何回复均取消：
```

重试次数上限：**3次**

---

## 审批开关

### 配置位置

`~/.verticals/config.json`

### 配置项

```json
{
  "APPROVAL_REQUIRED": true
}
```

| 值 | 说明 |
|---|------|
| `true` | 每次素材生成需要"同意"审批（默认） |
| `false` | 自动执行，跳过审批 |

### 使用场景

- **开发测试**：`false`，快速迭代
- **正式生产**：`true`，控制成本
- **批量生成**：`false`，无人值守

---

## 成本计算

### 图片成本

```
图片成本 = 图片数量 × $0.05
```

### 视频成本

```
视频成本 = 视频数量 × $0.2
```

视频时长仅支持 4/6/8 秒，成本相同。

### 总成本

```
总成本 = 图片成本 + 视频成本
```

---

## 注意事项

1. **模型锁定**：成本仅适用于当前模型，禁止切换模型（详见 `模型锁定说明.md`）
2. **审批必须文字**：语音、表情、ID审批一律无效
3. **重试也需审批**：每次重试都是新的成本，必须重新审批
4. **开关不影响显示**：即使关闭审批，仍会显示成本预估
