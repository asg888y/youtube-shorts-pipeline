## 新增功能 (2026-05-09)

### 热点平台扩展

**新增平台**：
- 抖音热点：挑战赛、爆款BGM、热门特效、情感语录等
- 快手热点：同城热点、才艺展示、三农内容、搞笑短剧等
- 小红书热点：探店分享、穿搭教程、护肤心得、家居好物等

**原有平台**：
- 微博热搜：最实时的社会热点
- 知乎热榜：深度讨论话题
- 百度热搜：综合热点

### 文案输入方式

**1. 直接输入文案**
```
文案：今天要给大家分享一个超级好用的工具...
script: 今天要给大家分享...
```
系统直接使用输入的文案，跳过热点获取和改写。

**2. 视频链接转录**
```
https://www.douyin.com/video/123456
https://www.bilibili.com/video/BV1xx411c7xx
```
系统自动下载音频，使用Whisper API转录语音为文字。

**3. 热点选择**
```
3
```
选择第3个热点，系统提取相关内容并改写。

### 链接内容提取

**支持平台**：
- 抖音：提取视频标题、描述、语音转录
- 快手：提取视频基本信息
- 小红书：提取笔记内容
- B站：提取视频标题、描述、语音转录
- YouTube：提取视频标题、描述、语音转录

**技术实现**：
- yt-dlp：视频信息提取
- Whisper API (8.138.165.244:80)：语音转文字
- requests：网页内容提取

### 相关文件

- `verticals/topics/__init__.py`：热点获取功能
- `verticals/link_extractor.py`：链接内容提取器
- `state_helper.py`：参数解析和状态管理

### Why

用户需要更灵活的文案输入方式，以及短视频平台的热点内容。新增功能支持直接输入文案、视频链接转录，以及抖音、快手、小红书的热点获取。

### How to apply

使用时，用户可以直接输入文案、粘贴视频链接，或选择热点编号。系统会自动识别输入类型并处理。
