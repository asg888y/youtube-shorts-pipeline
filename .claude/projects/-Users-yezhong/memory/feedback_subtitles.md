## 禁止

1. **字幕字号按视角比例 4.5-6% 帧高**。1080P下=49-65px。禁止 <4.5%（太小看不见）或 >6%（堆满屏幕）。
2. **禁止 Bold=1 + Outline > 2.0**。粗体+厚描边会产生视觉叠影。
3. **禁止单行超过合理字数**。必须按字数自动换行，中文每行≤10-12字。
4. **禁止每条脚本段只生成一条 cue**。必须拆分，每个 chunk 独立 cue。

## 正确配置

- `SUB_FONT_SIZE` = 4.5-6% 帧高（1080p → 49-65px）
- `Bold=0, Outline=1.5`
- 按字数自动换行，中文每行≤12字
- 每个 chunk 独立 SRT cue，时间均分

**Why:** 之前视频出现两层字幕叠加+堆满屏幕，排查确认是字号过大+粗体厚描边+长文本未拆分。

**How to apply:** 每次生成字幕前按视角比例计算字号，按字数拆行。
